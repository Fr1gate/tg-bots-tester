# Telegram Bot Tester

Минималистичный веб-интерфейс для ручного тестирования Telegram-ботов. Вводишь токен, выбираешь чат, отправляешь сообщение — бот пишет от своего имени.

## Быстрый старт

```bash
npm install
npm run dev
```

Открой `http://localhost:47891`.

## Использование

1. **Bot Token** — вставь токен от [@BotFather](https://t.me/BotFather)
2. **Resolve** — нажми, чтобы получить список чатов из истории обновлений бота (бот должен уже получить хотя бы одно сообщение от нужного пользователя/группы)
3. Выбери чат из списка или вбей Chat ID вручную
4. Введи сообщение → **Send** (или `Ctrl+Enter`)

## Как получить Chat ID

Напиши своему боту любое сообщение, затем нажми кнопку Resolve — появится список найденных чатов.


## Ограничения

- Бот может писать пользователю только если тот **первым написал боту** (ограничение Telegram)
